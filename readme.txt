== Description ==

1. Require Silverlight  install from here http://www.microsoft.com/silverlight/
2. Use the built-in Wordpress   shortcode [SilverPlayer url="http://youvideo"] 
3. for test  sample folder can test video and see screenshot

   

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `wp-silver-player` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
